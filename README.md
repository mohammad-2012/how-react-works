# ⚛️ How React Works

A simple project to understand React internals - how JSX gets converted to elements, how rendering works, and the magic behind the scenes.

## 🎯 What I Learned

- How JSX transforms to `React.createElement()`
- Virtual DOM vs Real DOM
- How React renders and re-renders components
- The power of keys in lists
- Component lifecycle
